# Crate Local-Doc Coverage

This page is a maintenance inventory for workspace-local context files used by
contributors and coding agents.

It tracks the crate-boundary triplet for each first-party crate:

- `README.md`
- `CLAUDE.md`
- `AGENTS.md`

It does not try to catalog every nested module doc in the tree. Those files
live next to the modules they describe and are reviewed in place.
