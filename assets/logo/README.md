# Shipper Logo Assets

This directory contains Shipper logo assets for README and project presentation use.

## Files

- `shipper-container-plain.svg` - default README logo.
- `shipper-container-plain-dark.svg` - dark variant.
- `shipper-container-plain-light.svg` - light variant.
- `shipper-container-plain-lockup.svg` - container mark plus wordmark lockup.
- `shipper-container-plain-preview-dark.png` - dark preview image.
- `shipper-container-plain-preview-light.png` - light preview image.

Use the SVG assets when possible. The PNG files are previews for surfaces that
do not render SVG well.
